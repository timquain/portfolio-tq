Artifact 1 • Prototype

JayTees Fabric Assistant

An AI concierge that helps shoppers find fabrics, estimate yardage, and understand policies—built with clarity, safety, and testability in mind.

Role: Solo build Stack: FastAPI · OpenAI · YAML tests Focus: Trustworthy UX
  • Objective: ship a safe, reliable assistant that reduces repetitive support and speeds up purchase decisions.
  • Process: Empathize → map intents → prototype (FastAPI + OpenAI) → smoke + scenario tests → tune prompts and cross‑sell logic.
  • Audience fit: concise copy, clarifying questions, and a verification runbook for reviewers and collaborators.

Highlights

  • Scenario and unit tests cover refunds, shipping, care, and safety guardrails.
  • Product catalog + routing rules produce grounded recommendations and FAQs.
  • Config + feedback endpoints make the prototype easy to tune in the wild.

Reflections

Significance: This work fuses human‑centered UX with backend rigor—every answer is concise, auditable, and designed for safe extensibility.

Revisions: Added branded UI concepts, runbook steps, and clearer error handling based on early reviewer feedback.

Artifact 3 • Research Story

From Molecules to Machine Learning

An interdisciplinary walkthrough that pairs computational chemistry instincts with neural networks and adaptive communication.

Role: Solo author Stack: Python · Quantum ESPRESSO · NumPy Focus: Translation between domains
  • Objective: show how ML concepts plug into molecular modeling and how to explain that bridge to mixed audiences.
  • Process: mapped research background → identified neural network touchpoints → rewrote for clarity using chunking, analogy, and scaffolding.
  • Audience fit: non-specialists get approachable narratives; scientists get concrete tooling and validation hooks.

Highlights

  • Use cases: adsorption prediction, STM conductance patterns, materials screening.
  • Communications toolkit: structured messaging, analogy-led explanations, and visuals.
  • Outcome: a reusable template for presenting interdisciplinary research with AI assist.

Reflections

Significance: Codifies how I translate lab insights into ML-ready problems while keeping rigor intact.

Lessons: Simplifying dense topics builds trust and accelerates collaboration across teams.

Artifact 4 • Technical Deep Dive

Deep Learning & Materials

Explores how graph neural nets and modern deep learning accelerate materials discovery, semiconductor analytics, and lab insights.

Role: Solo author Stack: TensorFlow · PyTorch · Scikit-learn Domains: MOFs · Adsorption · Semiconductors
  • Objective: connect neural architectures to real R&D tasks, surfacing where AI can augment physics-based methods.
  • Process: consolidated DFT and adsorption experience → mapped to deep learning approaches → captured validation and ethics guardrails.
  • Audience fit: balances scientific rigor with clear calls to action for data, metrics, and reproducibility.

Highlights

  • Applications: DFT approximation, adsorption prediction, STM trace analysis, and materials screening.
  • Modeling mix: feedforward nets, CNNs, and graph networks with domain-driven validation.
  • Outcome: a roadmap showing where AI accelerates simulations and decision-making.

Reflections

Significance: Bridges physics intuition with machine learning to make experimentation faster and smarter.

Revisions: Expanded scientific use cases and clarified how deep learning complements—not replaces—physical models.
