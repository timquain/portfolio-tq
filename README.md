# Timothy Quainoo - AI/ML Portfolio

This repository contains a static single-page portfolio for Timothy Quainoo's AI/ML work. The page presents seven artifacts across conversational AI, computational chemistry, machine learning communication, and leadership.

## Project Structure

- `index.html` contains the full portfolio page, styles, and small smooth-scroll behavior.
- `assets/` contains the portfolio visuals and downloadable source artifacts referenced by the page.

## Local Preview

Use a lightweight static server from the repository root so relative assets load correctly:

```bash
python3 -m http.server 8000
```

## Deploy to Netlify

This is a static site. Use no build command and publish the repository root.
